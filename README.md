# Техническое задание на разработку системы управления библиотекой (C#)

## 1. Введение

Система управления библиотекой предназначена для автоматизации процессов учета книг.
Она будет предоставлять функционал для добавления, удаления и поиска книг.

## 2. Цели и задачи

Цель: Создать консольное приложение на C# для управления библиотекой книг.
Задачи:

* Реализовать функционал для добавления, удаления и поиска книг.
* Реализовать обработку ошибок и валидацию данных.

## 3. Функциональные требования

### 3.1. Книги

Book:
Id  (уникальный идентификатор книги)
Title (название книги)
Author (автор книги)
Year (год издания)

Функциональность:

* Добавление новой книги (Library.exe add book -title Название книги -author Автор книги -Year 2003.
* Удаление книги по идентификатору (Library.exe delete book 1).
* Поиск книги по названию или автору (Library.exe find -name Название) (Library.exe find -Year 2003).
* Вывод списка всех книг (Library.exe books).

## 4. Нефункциональные требования

Приложение должно работать в консольном интерфейсе.
Данные о книгах должны сохраняться между сессиями в виде текстовых файлов (JSON).
Обработка ошибок должна включать проверку на корректность введенных данных. Например, отсутствие пустых полей.
Код должен быть разделен на модули (классы) для улучшения читаемости и поддержки.

## 5. Входные данные

Ввод данных азователем через консоль (например, название книги, и т.д.).

## 6. Выходные данные

Вывод результатов операций (например, подтверждения добавления книги или списка всех книг).

## 7. Интерфейс

Неинтерактивное взаимодействие с выводом результата операции:

Управление книгами:
1.1. Добавить книгу
1.2. Удалить книгу
1.3. Поиск книги по всем полям.
1.4. Вывести полный список книг.
1.5. Выход.

## 8. Тестирование

Провести тестирование функционала на корректность работы основных операций.
Проверить обработку ошибок (например, при вводе некорректных данных).

## 9. Документация

Подготовить краткое руководство пользователя.
Комментарии к коду для объяснения ключевых частей.
